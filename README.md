# 사주 기반 부캐명 추천 서비스

생년월일시로 사주를 분석하고, 개성있는 부캐명을 추천해주는 웹 서비스입니다.

## 기능

- 🔮 정확한 사주팔자 분석 (한국천문연구원 API 사용)
- 💫 오행 기반 특성 분석
- 🎮 두 가지 타입의 부캐명 추천
  - 💪 내 특성 살리기 (강점 극대화)
  - ⚖️ 부족한 기운 채우기 (균형 보완)
- 🇰🇷 한국풍 부캐명 (형용사형, 한자명 등)

## 기술 스택

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- API: 한국천문연구원 음양력 정보 API
- Deploy: Vercel

## 로컬 실행

```bash
npm install
npm start
```

http://localhost:3000 에서 확인
